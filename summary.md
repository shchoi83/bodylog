## 식단 Q&A 핵심 정리

**서브웨이 최적 조합 (확정)**
로티세리 치킨 샐러드 + 동일미트 추가 + 피클/올리브/치즈 제외 + 소금/식초/후추

**초밥 먹을 때**
간장/초생강 제외 / 와사비만 / 8~10피스 기준

**훈제연어 (트레이더스)**
노르웨이 훈제연어 트리플팩 150g×3 / 소스 제외 / 와사비만
→ 150g 기준: Cal≈276 / 단백질≈22~24g

**포도당 캔디 활용**
운동 시작 전 2~3개 + 중반 2개 / 훈련 중 실시간 ATP 공급

**WPH vs WPI 구분**
WPH = 운동 직후 골든타임 전용
WPI = 일반 보충 / 취침 전
